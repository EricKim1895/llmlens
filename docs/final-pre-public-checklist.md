# Final pre-public checklist

## Product

- [ ] 产品定位是否清晰
- [ ] Mention / Citation / Recommendation 是否容易理解
- [ ] Score 是否避免误导

## Technical

- [ ] API key 安全
- [ ] Access code 工作正常
- [ ] Error handling
- [ ] Export 功能
- [ ] Mobile layout
- [ ] Browser compatibility

## Cost

- [ ] Perplexity credits 风险
- [ ] 是否需要 rate limit
- [ ] 是否需要 daily quota

## Legal / Trust

- [ ] Disclaimer
- [ ] Privacy page
- [ ] Terms page（如果公开）

## SEO / Release

- [ ] 是否继续 noindex
- [ ] 是否准备公开测试
- [ ] 是否邀请小范围用户
