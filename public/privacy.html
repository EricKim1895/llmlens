Privacy Notice

Mock mode

Mock audits run with deterministic sample data. Mock inputs are not sent to Perplexity, and Mock mode does not consume Perplexity API credits.

Perplexity mode

For a live Perplexity audit, your brand, website, industry, competitors, and generated prompts may be sent through the LLM Lens serverless endpoint to Perplexity to produce the audit. Do not submit confidential, personal, regulated, or other sensitive information.

Data storage

LLM Lens does not currently provide user accounts. Audit inputs and results are not intentionally stored in an LLM Lens application database. You can download results locally through Markdown or JSON export. Vercel and Perplexity may process technical or request data according to their own services and policies.

Access code

The beta access code is used only to limit access to live Perplexity audits. It is not an account password and should not be shared publicly.

Diagnostic disclaimer

Results are estimates for diagnostic review. They are not rankings, traffic forecasts, or guarantees that an AI system will cite or recommend a brand.
